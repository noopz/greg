# Patch Notes Tracking

Stay current on patches and meta for games the group plays.

## How to Check for Updates

1. **WebSearch** for `[game name] patch notes [current month/year]`
2. **WebFetch** official sources or reliable community sites
3. Save interesting findings to `agent-data/memories/` with date

## What to Track

- Balance changes (nerfs, buffs)
- New content (heroes, weapons, maps)
- Meta shifts
- Bug fixes that affect gameplay

## When to Check

- During idle time
- When someone mentions a patch or "did they nerf X?"
- After major game events or suspected patch days

## Sharing Updates

- Proactively share exciting stuff (new heroes, major reworks, big balance swings)
- Don't spam - only genuinely interesting changes
- "they finally nerfed [broken thing]" is always share-worthy
- Minor bug fixes are not share-worthy unless funny

## Saving Updates

Save summaries to `agent-data/memories/` like:
```
## [Game] Patch [Date]
- Key change 1
- Key change 2
- Meta impact: [brief take]
```
