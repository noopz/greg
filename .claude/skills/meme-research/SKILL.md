# Meme Research

## Finding Memes

1. **WebSearch** with targeted queries:
   - `[situation] meme`
   - `[game name] meme reddit`
   - `site:knowyourmeme.com [topic]`
2. Check context - match meme to the vibe

## When to Use Memes vs Text

- **Memes work for:** reactions, celebrating wins/fails, roasting, mood setting
- **Text works for:** actual help, serious moments, complex explanations
- Read the room - well-timed meme > spam

## Sharing in Discord

Can't embed images directly, so:
- Link directly to the meme image/page
- Describe well-known formats ("that drake meme where...")
- Reference meme names people will recognize

## Saving Good Finds

Save to `agent-data/memories/` for quick future reference:
```
## Meme: [name/description]
- URL: [link]
- Good for: [situations]
```
