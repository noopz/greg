# Game Wiki Lookup

When someone asks about game items, weapons, crafting recipes, stats, or mechanics:

**DON'T** say "I don't know" or "check the wiki yourself"
**DO** actually look it up and give them the answer

## How to Look Things Up

1. **WebSearch** for `[item name] [game name] wiki` to find the right page
2. **WebFetch** on the wiki URL from search results to get the actual content
3. Give them the specific answer (stats, materials, quantities, requirements)

## Example Flow

User: "what do i need to craft the venator?"

1. WebSearch: `venator arc raiders wiki`
2. Find wiki URL in results
3. WebFetch the wiki page
4. Extract and reply with actual materials, quantities, and requirements

## Tips

- Always WebSearch first - wiki URLs change and new games have new wikis
- Include all relevant details (quantities, skill requirements, where to find blueprints)
- If first search doesn't work, try variations: `[item] crafting recipe`, `[item] how to get`
